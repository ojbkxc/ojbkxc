Hexo|Sakura主题

点击Sakura 下载主题，也可以到Hexo的主题库中找一个喜欢的主题。如果你不想太折腾可以下载wallleap大神的美化好的密码wallleap。

下载之后可以安装依赖

npm install

关于Sakura主题修改网上已存有很多了，这里推荐 yremp大神的

评论区QQ头像配置

下载valine.min.js我用的是1.3.10

把放到Hexo\themes\Sakura\source的js目录下
到Hexo\themes\Sakura\layout_partial`footer.ejs`内将

<script src='//unpkg.com/valine/dist/Valine.min.js'></script>

替换为

<script src='/js/Valine1.3.10.min.js'></script>

将

var C=function(e,n,r){
var i=u.create("div",{
class:"vcard",id:e.id}),
o=m.hide?"":'<img class="vimg" 
src="'+(m.cdn+a(e.get("mail"))+m.params)+'">',

替换为

 var C=function(e,n,r){
    var qq_img=m.cdn+a(e.get("mail"))+m.params;//默认gravator头像接口
     if(e.get("mail").indexOf("@qq.com") >= 0){
        var prefix = e.get("mail").replace(/@.*/, "");//前缀
        var pattern=/^\d+$/g;  //正则表达式，数字
        var result= prefix.match(pattern);//match 是匹配的意思
        if(result!==null){
            qq_img = "//q1.qlogo.cn/g?b=qq&nk="+ prefix +"&s=100";
        }
    }
    var i=u.create("div",{class:"vcard",id:e.id}),o=m.hide?"":'<img class="vimg" src="'+ (qq_img)+'">',

然后评论区有QQ邮箱的话就会显示其QQ头像
当然如果你真的不好找的话可以直接下载我这个改好的valine.min.js不过要将提到的Hexo\themes\Sakura\layout_partial\footer.ejs内的Valine1.3.10.min.js的版本号去掉，当然你也可以直接将footer.ejs内

<script src='//unpkg.com/valine/dist/Valine.min.js'></script>

替换为

<script src='https://cdn.jsdelivr.net/gh/ojbkxc/cdn/js/Valine.min.js'></script>

直接引用

关于评论进阶邮箱提醒和微信还是有空在加把吧
